## Features

- Easy to use.
- Lightweight with no heavy dependencies.
- Supports popular languages.

## Supported Languages

- **English** (`en`)
- **French** (`fr`)
- **Spanish** (`es`)
- **German** (`de`)
- **Italian** (`it`)
- **Portuguese** (`pt`)

## Installation

Install OssamaAourfi using npm:

```bash
npm install ossama-aourfi
```
